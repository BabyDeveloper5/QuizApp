# Quiz App

A simple multiple-choice Quiz App built using Kotlin and Jetpack Compose.

---

## Features

- Multiple-choice questions
- Score tracking
- Simple and modern UI
- Jetpack Compose UI
- MVVM architecture
- Local question data (no API yet)

---

## Tech Stack

- Kotlin
- Jetpack Compose
- MVVM
- State management with ViewModel
